# Анализатор сложности текста

Проект для анализа читаемости и синтаксической сложности русского текста.

## Функциональность

- Расчет средней длины предложений в словах
- Расчет среднего количества слогов в словах
- Вычисление общего балла сложности по формуле Флеша
- Определение уровня сложности текста (от 5 класса до университета)
- Максимальная глубина синтаксического дерева
- Количество придаточных предложений
- Максимальное расстояние между зависимыми словами
- Среднее количество дочерних узлов в дереве

## Установка

```bash
pip install syntax-complexity-analyzer
```

## Использование

### Запуск пакета
После установки введите в командной строке:
```bash
analyze-complexity
```
Программа запросит текст для анализа и отобразит результаты. 

## Уровни сложности

- **Очень сложно** (0-30 баллов) - эксперт
- **Сложно** (30-50 баллов) - университет  
- **Довольно сложно** (50-60 баллов) - старшие классы
- **Стандартно** (60-70 баллов) - 8-9 класс
- **Довольно легко** (70-80 баллов) - 7 класс
- **Легко** (80-90 баллов) - 6 класс
- **Очень легко** (90+ баллов) - 5 класс

## Читаемость
Оценивается по формуле Флеша–Кинкейда для автоматизированного анализа читабельности, адаптированной для русского языка: FRE = 206,835 − 1,52 × ASL − 65,14 × ASW, где ASL — средняя длина предложения, а ASW — среднее количество слогов в слове.
