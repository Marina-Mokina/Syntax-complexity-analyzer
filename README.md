# Syntax Complexity Analyzer

Сервис для анализа читабельности и синтаксической сложности русского текста.

## Функциональность

- Расчёт средней длины предложений в словах
- Расчёт среднего количества слогов в словах
- Вычисление общего балла сложности по формуле Флеша
- Определение уровня сложности текста Флеша-Кинкейда
- Поиск текста в корпусе открыток (определение номера, даты и тега, если текст найден)
- Синтаксический анализ с помощью библиотеки Natasha:
  - Максимальная глубина синтаксического дерева
  - Количество придаточных предложений
  - Максимальное расстояние между зависимыми словами
  - Среднее количество дочерних узлов в дереве
  - Визуализация деревьев зависимостей

## Установка и использование

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/syntax-complexity-analyzer.git
cd syntax-complexity-analyzer
```

2. Установите зависимости через Poetry:
```bash
poetry install
```

3. Создайте пакет
```bash
poetry build
```

Пакет будет создан в папке `dist/`:
- `syntax_complexity_analyzer-0.1.0-py3-none-any.whl` (wheel-файл)
- `syntax_complexity_analyzer-0.1.0.tar.gz` (исходный код)

4. Установите пакет
```bash
poetry install
```

5. Запустите программу
- после установки пакета (из любой папки)
```bash
syntax_complexity_analyzer
```
- напрямую из исходного кода
```bash
poetry run python src/syntax_complexity_analyzer/main.py
```

## Уровни сложности

- Очень сложно (0–30 баллов) — эксперт
- Сложно (30–50 баллов) — высшее образование
- Довольно сложно (50–60 баллов) — старшие классы
- Стандартно (60–70 баллов) — 8–9 класс
- Довольно легко (70–80 баллов) — 7 класс
- Легко (80–90 баллов) — 6 класс
- Очень легко (90+ баллов) — 5 класс

## Читаемость

Оценивается по формуле Флеша—Кинкейда для автоматизированного анализа читабельности, адаптированной для русского языка:

FRE = 206,835 − 1,52 × ASL − 65,14 × ASW

где:
- ASL — средняя длина предложения (в словах)
- ASW — среднее количество слогов в слове
